# agent-team-blog

> 一个由 AI Agent 团队协作共建的个人博客，无任何人工代码介入，纯靠自然语言驱动。

🔗 **博客地址：[https://fatwill.cloud](https://fatwill.cloud)**

---

## ✨ 项目简介

这是一个特别的博客项目——从第一行代码到每一次功能迭代，全部由 AI Agent 团队以自然语言协作完成，**没有任何人工直接编写代码**。

整个博客由三个 Agent 分工协作、持续维护：

| Agent | 职责 |
|-------|------|
| 🖥️ **前端 Agent** | 负责 Nuxt 3 前端开发，包括页面、组件、交互与样式实现 |
| ⚙️ **后端 Agent** | 负责 Go + Gin 后端开发，包括 API 接口、数据建模与业务逻辑 |
| ✍️ **写手 Agent** | 负责博客文章的搜集、撰写与发布，生成封面图与配图 |

所有需求由人类以**自然语言**描述，经 PM（项目经理 Agent）拆解分发，各 Agent 独立完成开发、构建与部署，全程无人工代码介入。

---

## 🛠️ 技术栈

- **前端**：Nuxt 3 + Vue 3 + TypeScript + Tailwind CSS + Ant Design Vue
- **后端**：Go + Gin + SQLite + Redis
- **部署**：Nginx + systemd，腾讯云 COS 对象存储
